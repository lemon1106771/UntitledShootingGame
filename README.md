# Untitled Shooting Game 

A fast-paced, 2D platform shooter built in Python using Pygame. Created as a custom code project for COS10009, this game features chaotic arena combat inspired by titles like *Stick Fight: The Game* and *Ultrakill*. Survive endless waves of enemies, manage your ammo, and climb the leaderboard!

## Installation & Setup

1. Make sure you have [Python](https://www.python.org/downloads/) installed on your computer.
2. Install the Pygame library by opening your terminal or command prompt and running:
   ```bash
   pip install pygame
